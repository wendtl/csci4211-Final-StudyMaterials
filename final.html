LAN Protocols

Ethernet Token Ring
Better for less congestion. Hosts do not have to wait for token to send. Designed for high traffic
CSMA & CSMA/CD Uses 3-byte token, passed around 'ring' of connected devices
Forward all traffic it receives to next member in ring until it gets token. Maximum bound is set on wait time.

Media Access Control (MAC) Protocols

CSMA/CA CSMA/CD
Waits for medium to not be in use before transmitting Listens before and during transmission to make sure nobody is transmitting. If collision is detected, transmits a jam signal so others know there was a collision
more common in wireless networks. Wireless cant detect collision so it must use CA to avoid them. used to be more common in wired networks

EGP routes between sets of Autonomous Systems (AS). (BGP is a form of EGP). The "glue" of the internet

BGP directs traffic between Autonomous Systems. Sends updated router table information only when something changes. Diagram

Basic overview of OSI Model

Layer Description
Application Layer Protocols like HTTP, SMTP, FTP
Presentation Layer Protocols like JPEG, MPEG
Session Layer Protocols like NFS, SQL, RPC
Transport Layer Protocols like TCP, UDP
Network Layer Protocols like IP, ICMP
Data Link Layer Protocols like HDLC, ppp
Physical Layer The physical wire that data travels on

Transport Layer

  • multiplexing and demultiplexing
  • UDP & TCP
  • Stop&Wait, Go-Back-N, selective repeat

Network Layer

  • End to end data delivery: addressing, routing, forwarding
  • IP Addresses

Data Link Layer

  • 802.11b Wireless LAN (receiver ACK needed because of hidden terminal problem: A transmitter cant tell who else is transmitting so it must use Collision Avoidance.)
  • 802.11b Wireless LAN tends not to use CA if sending a very small amount of data
    • More on hidden/exposed terminal: Request to send (RTS) & Clear to send (CTS)
  • PPP: point to point link layer protocol
  • repeaters, switches, routers

Network Control Plane

  • intra-domain routing algorithms
  • SDN controllers (OSPF, RIP, OpenFlow, ODL, ONOS)
